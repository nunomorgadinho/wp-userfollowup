== Description ==

UserFollowUp connects events triggered by your customers to automatic follow up emails sent in your name, automating common tasks, reminding them of incomplete actions, brigging back customers and creating better service.

== Installation ==

= Automatic =

1. In the admin panel plugins page click Add New
2. Search for WP UserFollowUp
3. Find WP UserFollowUp in the list an click Install Now
4. Click OK when prompted
5. Activate the plugin
6. Done!

= Manual =

1. Extract wp-userfollowup.zip to your `wp-content/plugins` directory.
2. In the admin panel under plugins activate WP UserFollowUp.
3. Done!

= Configuration =

1. Visit the newly created entry for WP UserFollowUp on your dashboard's Settings menu
2. Enter your userfollowup tracking ID
3. Save your settings
